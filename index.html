Regulier of Versneld
4 jaar naar 3 jaar

In deze opleiding is het mogelijk om binnen 3 jaar af te studeren. Als je goed meedoet, extra opdrachten uitvoert en actief deelneemt aan open dagen, en daarnaast alle toetsen en projecten haalt, kun je ervoor kiezen na een halfjaar het versnelde traject te volgen.

Het Rooster

Als je in de eerste half jaar zit, ik zal zo de vakken laten zien wat ze inhouden.

Dit Rooster is als voorbeeld het kan dat je een ander rooster krijgt maar dit is ook allen een voorbeeld hoe het er uit kan gaan zien.

Wat vind ik van de opleiding

Bij de start van de opleiding had ik alleen enige ervaring met Python en HTML, maar ik wist niet goed wat ik kon verwachten. De opleiding heeft echter duidelijk uitgelegd hoe alles werkt. Bijvoorbeeld, bij de native les begin je met Microbit om de basisprincipes van Python te leren. Bij de web les had ik Elton als docent, die de stof uitstekend uitlegde. Hij gaf ons veel voorbeelden en tijd om zelf dingen te maken, waarbij we altijd vragen konden stellen en hulp kregen waar nodig.

Welke vakken heb je met uitleg.

Web HTML , CSS

Bij de web leer je vooral hoe je websites kunt stylen. Tijdens de lessen krijg je als eindopdracht de taak om een website na te maken op basis van een ontwerp met alleen de plaatjes en een voorbeeld van hoe het eruit moet zien.

Native

Bij de native begin je de eerste paar weken met het werken met Microbit om te begrijpen hoe het werkt. Je leert de basisprincipes van programmeren door het gebruik van Microbit. Nadat je de basis onder de knie hebt, ga je verder met Python. Als eindopdracht moet je een aantal projecten in Python maken

Praktijk

Tijdens de praktijklessen leer je hoe de werkpraktijk eruitziet en met welke systemen je te maken zult krijgen. Je maakt kennis met verschillende werkmethoden, zoals team soorten, en andere relevante systemen en processen.

PRO

Bij de PROlessen wordt vooral de Scrum-methodiek uitgebreid uitgelegd, evenals de basisprincipes van GitHub. Je leert hoe je deze tools effectief kunt gebruiken in een professionele omgeving
